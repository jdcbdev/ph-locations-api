# Philippines Locations API

REST API for serving Cities, Barangays, Provinces and Regions in Philippines

## Requirements
* NodeJS 10 or higher
* MySQL 7.5 or higher
* Web Server (Apache or Nginx)

## Installation

install dependencies
```
npm install
```

configure
```
cp .env.example .env
```

migrate
```
adonis migration:run
```

seed
```
adonis seed
```
